# 钢材表面缺陷智能识别与分析平台V1.0 使用说明

## 系统概述

本平台以深度学习与 YOLO 目标检测技术为核心，通过 `best.pt` 模型对钢材表面缺陷进行自动识别、统计和可视化分析。系统支持图片检测、视频检测、实时摄像头检测、历史记录查询、统计报表、用户权限、系统设置和文件管理。

## 角色权限

- 超级管理员：用户、角色、系统设置、日志、检测与文件全部权限。
- 检测员：图片/视频/摄像头检测、记录查询、统计分析、文件访问。
- 普通用户：基础检测和本人记录查询。

## 功能说明

### 首页大屏

展示累计检测、累计缺陷、今日检测、今日缺陷、近 7 天趋势、缺陷类别分布和最近检测记录。

### 图片检测

上传钢材表面图片，系统调用 YOLO 模型输出缺陷类别、置信度、坐标框，并生成带标注结果图。

### 视频检测

上传生产线视频，系统按帧抽样检测并生成结果视频，同时汇总缺陷数量和类别信息。

### 摄像头检测

使用浏览器摄像头实时采集画面，定时上传帧到后端检测，并在画布上实时绘制缺陷框。

### 检测记录

支持按检测类型、缺陷类别筛选记录，查看详情、删除记录，并通过文件管理下载结果图或结果视频。

### 统计分析

通过柱状图、折线图展示缺陷类别分布和质量趋势，为钢材生产质量追溯和工艺优化提供依据。

### 系统管理

管理员可维护用户、角色权限、系统参数，并查看登录日志与操作日志。
